============================================================
   BUCH-ANZEIGEN-HELFER  –  ANLEITUNG (Windows)
============================================================

Dieses Programm macht aus Fotos eines Buches eine fertige
eBay-Anzeige. Du musst dich nicht auskennen – folge einfach
den Schritten der Reihe nach. Lass dir ruhig Zeit.

Du brauchst:
  - einen Windows-Computer mit Internet
  - den Zugangs-Schlüssel (bekommst du von Giacomo)
  - wenn du mit dem Handy fotografieren willst:
    Handy und Computer im gleichen WLAN

Tipp: Nimm dir die Schritte einzeln vor. Nach der ersten
Einrichtung genügt später ein Doppelklick.


============================================================
   TEIL 1 – EINMALIG EINRICHTEN
============================================================

SCHRITT 1: Den Ordner auspacken
-------------------------------
Du hast eine Datei mit der Endung ".zip" bekommen.
  1. Mit der rechten Maustaste auf die ZIP-Datei klicken.
  2. Im Menü "Alle extrahieren..." anklicken.
  3. Auf "Extrahieren" klicken.
Es entsteht ein Ordner. Darin liegt eine Datei namens
"Start.bat" – die brauchst du gleich.


SCHRITT 2: Python installieren (nur beim allerersten Mal)
---------------------------------------------------------
Das Programm benötigt ein Hilfsprogramm namens "Python".
Es ist kostenlos.

  1. Auf "Start.bat" doppelklicken.
  2. Ist Python noch nicht da, öffnet sich von selbst eine
     Internetseite. Dort den großen Knopf "Download Python"
     anklicken und die heruntergeladene Datei öffnen.
  3. GANZ WICHTIG: Im ersten Fenster unten ein Häkchen setzen
     bei  "Add python.exe to PATH".
     (Ohne dieses Häkchen funktioniert es nicht.)
  4. Dann auf "Install Now" klicken und warten, bis "Setup
     was successful" erscheint. Fenster schließen.

Ist Python schon vorhanden, kannst du diesen Schritt
überspringen.


SCHRITT 3: Das Programm zum ersten Mal starten
----------------------------------------------
  1. Wieder auf "Start.bat" doppelklicken.
  2. Es öffnet sich ein schwarzes Fenster. Beim ersten Mal
     richtet sich das Programm ein bis zwei Minuten lang
     selbst ein. Das ist normal – einfach warten.
  3. Danach öffnet sich von allein dein Internet-Browser
     mit dem Programm.

WICHTIG: Das schwarze Fenster bitte NICHT schließen,
solange du arbeitest. Es ist der "Motor" im Hintergrund.
Zum Beenden schließt du es einfach (oder klickst im
Programm unten auf "Programm beenden").


SCHRITT 4: Den Schlüssel eintragen (nur einmal)
-----------------------------------------------
Damit die künstliche Intelligenz arbeiten darf, trägst du
einmal den Zugangs-Schlüssel ein.
  1. Im Programm oben rechts auf "Einstellungen" klicken.
  2. Beim "Anthropic-API-Schlüssel" den Schlüssel eintippen
     oder einfügen, den dir Giacomo geschickt hat.
  3. Auf "Speichern" klicken.

Geschafft! Die Einrichtung ist fertig.


============================================================
   TEIL 2 – SO BENUTZT DU ES AM COMPUTER
============================================================

  1. Auf "Start.bat" doppelklicken (Browser öffnet sich).
  2. Fotos des Buches auf den Computer holen und mit
     "Fotos auswählen" hinzufügen.
  3. Auf "Anzeige erstellen" klicken und kurz warten.
  4. Text prüfen, bei Bedarf "Preis recherchieren",
     dann zur eBay-Datei hinzufügen.


============================================================
   TEIL 3 – MIT DEM HANDY FOTOGRAFIEREN (am bequemsten)
============================================================

So kannst du das Buch direkt mit dem Handy fotografieren –
ganz ohne Bilder umständlich zu übertragen.
Handy und Computer müssen im gleichen WLAN sein.

  1. Am Computer auf "Start.bat" doppelklicken.
  2. Im schwarzen Fenster erscheint ein QR-Code (ein
     schwarz-weißes Quadrat-Muster).
  3. Nimm das Handy und öffne die Kamera. Halte sie auf den
     QR-Code auf dem Bildschirm – es erscheint ein Link.
     Diesen Link antippen.
  4. Das Programm öffnet sich jetzt im Handy.
  5. Auf "Fotos auswählen" tippen. Das Handy fragt, ob du
     fotografieren oder ein vorhandenes Foto nehmen willst.
     Fotografiere das Buch (oder wähle ein Foto aus).
  6. Auf "Anzeige erstellen" tippen.

Klappt der QR-Code nicht? Im schwarzen Fenster steht direkt
darüber auch eine Adresse wie  http://192.168.x.x:5050 .
Diese kannst du im Browser des Handys von Hand eingeben.


============================================================
   WENN WINDOWS NACHFRAGT (ist harmlos)
============================================================

Beim ersten Start können ein bis zwei blaue oder
Hinweis-Fenster erscheinen. Das ist normal:

  - "Der Computer wurde durch Windows geschützt":
    Auf "Weitere Informationen" klicken,
    dann auf "Trotzdem ausführen".

  - Windows-Firewall / "Zugriff zulassen":
    Auf "Zugriff zulassen" klicken.
    (Nur so kann das Handy den Computer im WLAN erreichen.)


============================================================
   WENN ETWAS KLEMMT
============================================================

  - Erster Tipp, hilft fast immer: das schwarze Fenster
    schließen und "Start.bat" noch einmal doppelklicken.

  - Steht im schwarzen Fenster eine Fehlermeldung in rot?
    Mach ein Foto vom Bildschirm und schicke es Giacomo.

  - Handy findet das Programm nicht? Prüfe, ob Handy und
    Computer im gleichen WLAN sind.

Viel Erfolg!
