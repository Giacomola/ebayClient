============================================================
   BUCH-ANZEIGEN-HELFER  –  ANLEITUNG (Windows)
============================================================

Dieses Programm macht aus Fotos eines Buches eine fertige
eBay-Anzeige. Du musst dich nicht auskennen – folge einfach
den Schritten der Reihe nach. Lass dir ruhig Zeit.

Die Einrichtung hat Giacomo schon für dich erledigt. Auf dem
Desktop (Startbildschirm) liegt ein Symbol mit dem Namen
"Buch-Anzeigen-Helfer". Das ist alles, was du brauchst.

Du brauchst:
  - den Computer mit Internet
  - wenn du mit dem Handy fotografieren willst:
    Handy und Computer im gleichen WLAN


============================================================
   TEIL 1 – SO BENUTZT DU ES AM COMPUTER
============================================================

  1. Auf dem Desktop auf das Symbol "Buch-Anzeigen-Helfer"
     doppelklicken. Dein Internet-Browser öffnet sich von
     selbst mit dem Programm. (Ein schwarzes Fenster erscheint
     NICHT – das ist richtig so.)

  2. Fotos des Buches auf den Computer holen und mit
     "Fotos auswählen" hinzufügen.
     (Bequemer geht es mit dem Handy – siehe Teil 2.)

  3. Auf "Anzeige erstellen" klicken und kurz warten
     (etwa eine halbe bis eine Minute). Die künstliche
     Intelligenz schreibt den Text und sucht Beispielpreise.

  4. Text in Ruhe durchlesen und bei Bedarf anpassen.
     Der vorgeschlagene Preis steht schon im Preisfeld.

  5. Auf "Zur eBay-Sammeldatei hinzufügen" klicken.
     Das Buch ist damit gespeichert. Du kannst gleich das
     nächste Buch machen.

  6. Zum Beenden unten auf der Seite auf "Programm beenden"
     klicken. (Es gibt kein Fenster zum Schließen, weil das
     Programm unsichtbar im Hintergrund läuft.)


============================================================
   TEIL 2 – MIT DEM HANDY FOTOGRAFIEREN (am bequemsten)
============================================================

So fotografierst du das Buch direkt mit dem Handy – ganz ohne
Bilder umständlich zu übertragen.
Handy und Computer müssen im gleichen WLAN sein.

  1. Am Computer das Programm öffnen (Teil 1, Schritt 1).
  2. Auf den Knopf "📱 Per Handy hochladen" klicken.
     Es erscheint ein QR-Code (ein schwarz-weißes
     Quadrat-Muster) und darunter eine Adresse.
  3. Nimm das Handy und öffne die Kamera. Halte sie auf den
     QR-Code auf dem Bildschirm – es erscheint ein Link.
     Diesen Link antippen.
  4. Das Programm öffnet sich jetzt im Handy.
  5. Auf "Fotos auswählen" tippen. Das Handy fragt, ob du
     fotografieren oder ein vorhandenes Foto nehmen willst.
     Fotografiere das Buch (oder wähle ein Foto aus).
  6. Die Fotos erscheinen von selbst auch am Computer.
     Dort dann wie gewohnt auf "Anzeige erstellen".

Klappt der QR-Code nicht? Tippe die angezeigte Adresse
(etwa  http://192.168.x.x:5050 ) im Browser des Handys
von Hand ein.


============================================================
   WENN WINDOWS NACHFRAGT (ist harmlos)
============================================================

Beim allerersten Start kann ein Hinweis-Fenster erscheinen.
Das ist normal:

  - Windows-Firewall / "Zugriff zulassen":
    Auf "Zugriff zulassen" klicken.
    (Nur so kann das Handy den Computer im WLAN erreichen.)


============================================================
   WENN ETWAS KLEMMT
============================================================

  - Erster Tipp, hilft fast immer: das Programm einmal
    "Programm beenden" und das Desktop-Symbol erneut
    doppelklicken.

  - Die Seite ist leer oder öffnet sich nicht? Im Browser
    von Hand  http://127.0.0.1:5050  eingeben.

  - Handy findet das Programm nicht? Prüfe, ob Handy und
    Computer im gleichen WLAN sind.

  - Etwas anderes klemmt? Mach ein Foto vom Bildschirm und
    schicke es Giacomo.

Viel Erfolg!
